Java installieren

Es kann vorkommen, dass man Java benötigt, beispielsweise weil eine andere Software dies benötigt. Die Installation ist jedoch etwas komplizierter, wie man bei wiki.ubuntuusers nachlesen kann. Ich selbst habe mich für die Variante entschieden, bei der man Java von „webupd8team“ herunterlädt.

Die Vorgehensweise unterscheidet sich zwischen Ubuntu 16.04 und ElementaryOS 0.4

Ubuntu 16.04

Unter Ubuntu 16.04 konnte ich Java in der Version 8 mit folgenden Befehlen erfolgreich installieren:

  • sudo add-apt-repository ppa:webupd8team/java

  • sudo apt-get update

  • sudo apt-get install oracle-java8-installer

Mit dem ersten Befehl wird das Repository von "webupd8team" den Paketquellen hinzugefügt. Mit dem zweiten Befehl werden die Paketquellen dann abgefragt und die Liste der zur Verfügung stehende Software aktualisiert. Der dritte Befehl schließlich startet die Installation.

ElementaryOS 0.4

Da ich auch ElementaryOS einsetze, dachte ich, das geht genauso. Ist ja schließlich auch ein Ubuntu. Stimmte aber nicht…

  • sudo apt edit-sources

    um Zugriff auf die Datei zu erhalten, in welcher die Paketquellen verwaltet werden. Beim folgenden Dialog muss man sich dann einen Texteditor aussuchen

  • Am Ende der nun geöffneten Datei fügte ich nun

    deb http://ppa.launchpad.net/webupd8team/java/ubuntu xenial main

    ein und speicherte die Datei anschließend

  • sudo apt-get update

    um die Liste der zur Verfügung stehenden Pakete zu aktualisieren

  • sudo apt-get install oracle-java8-installer

Hinweis: Bei mir kam dann eine Fehlermeldung, wonach die Signatur nicht überprüft werden konnte, die Installation lief jedoch ohne Probleme.
