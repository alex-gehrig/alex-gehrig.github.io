Ubuntu in VirtualBox ohne GUI installieren

Virtuelle Maschinen sind eine praktische Sache und so kann es vorkommen, dass man sich eine solche in Form von Ubuntu installiert, aber ohne GUI. Im Anschluss stellt sich dann die Frage: wie kann man die Erweiterungen installieren, damit man beispielsweise "gemeinsame Ordner" nutzen kann, um Daten zwischen Host und Gast auszutauschen?

Wie so oft ist es eigentlich gar nicht so schwer - wenn man weiß was man tun muss:

  • Virtuelle Maschine starten und sich anmelden
  • sudo apt-get install virtualbox-guest-additions-iso
  • das heruntergeladene iso-Image liegt nun unter /usr/share/virtualbox/VBoxGuestAdditions.iso
  • sudo mkdir /mnt/cdrom
  • sudo mount /usr/share/virtualbox/VBoxGuestAdditions.iso /mnt/cdrom
  • sudo sh ./VBoxLinuxAdditions.run

Mit dem ersten Befehl lädt man die Erweiterungen herunter. Mit dem zweiten Befehl erstellt man sich ein Verzeichnis, in welches man mit dem dritten Befehl das iso-Image anschließend einbindet. Nun muss man lediglich nich dorthin navigieren und kann dann mittels des vierten und letzten Befehls die Installation starten.

Anschließend muss man bloß noch die Installation abnicken und abwarten.
